توضیحات

برد X7 Server یک گیت‌وی هوشمند برای اینترنت اشیاء (IoT Gateway) است که امکان اتصال تجهیزات صنعتی و ساختمانی به شبکه اینترنت و پروتکل‌های ارتباطی مختلف را بدون نیاز به IP استاتیک فراهم می‌سازد.

ویژگی‌ها

  • دو پورت RS485 برای ارتباط Modbus RTU
  • درگاه اترنت (Ethernet) برای اتصال مستقیم به شبکه
  • درگاه USB (برای توسعه‌های آتی)
  • پشتیبانی از پروتکل MQTT برای ارسال داده به سرورهای ابری
  • قابلیت Access Point داخلی با آدرس 192.168.4.1
  • تغذیه: 9–24V DC، جریان 1A
  • نصب روی ریل تابلو برق

کاربردها

  • گیت‌وی ارتباطی برای تجهیزات صنعتی و ساختمانی
  • مانیتورینگ و کنترل چیلرها، هواسازها، فن‌ها و پمپ‌ها
  • انتقال داده به سرورهای ابری بدون نیاز به IP استاتیک
  • اتوماسیون صنعتی و BMS
